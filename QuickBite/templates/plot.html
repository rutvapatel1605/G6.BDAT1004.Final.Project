Project Information

Project Title: Nexus Analytica - Data Integration and Visualization Solution

Project Completion Date: 12/14/2023

Client: QuickBites

Objective

The objective of this project is to conduct a comprehensive analysis of food establishments in two major cities, Mumbai and Delhi, using a provided dataset. The analysis focuses on key aspects such as home delivery services, average cost for two people, and the distribution of restaurant ratings.

Data Overview:

The dataset contains information about various food establishments, including their names, ratings, discount percentages, cashback amounts, cost for two people, city, locality, home delivery availability, and menu item counts.

Team and Roles

  • Developers (Rutva Patel): Led the development of the software, both front and back end.
  • Project Manager (Mitali Dhadich): Managed project deliverables, collaborated with the Business Analyst for documentation.
  • Business Analyst (Rahul Pandey): Developed comprehensive documentation for the proposal.
  • Data Analyst (Chintan Chauhan): Acquired data, built data models, and set up the database as needed.

Key Achievements

Data Acquisition

  • Successfully acquired data from a remote source (URL) using Python.
  • Utilized a meaningful dataset that demonstrated an understanding of XYZ's interests.

Database Management

  • Implemented a cloud database (MongoDB) for efficient storage.
  • Executed a pseudo-batch process running every 24 hours to acquire and store data.

Web Application Development

  • Developed a robust web application using the Flask framework (Python on the server-side).
  • Leveraged JavaScript for client-side development.
  • Displayed data through interactive charts and graphs using Google Charts.
  • Conducted a fit-gap analysis to ensure the application met QuickBites's requirements.
